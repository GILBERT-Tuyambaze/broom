# Broom Presentation Notes

This folder contains the project presentation for `Broom`.

## Files

- `Broom_Project_Presentation.pptx`: the PowerPoint deck for presenting the project.
- `../scripts/generate-project-presentation.mjs`: the generator script used to build or refresh the deck.

## Presentation structure

1. Title and project identity
2. Problem and opportunity
3. Solution overview
4. Current MVP features
5. Project strengths
6. Best features
7. Uniqueness
8. Audience and impact
9. Roadmap
10. Closing message

## Important presentation note

The deck is intentionally honest:

- It highlights features that already exist in the current app.
- It separates future ideas like mentorship, community, and growth content into the roadmap.
- It presents `Broom` as both a usable product today and a scalable long-term platform.
- It now includes seed-driven slides for community safety, mentorship verification, and AI guardrails.

## Regenerate the deck

Run:

```bash
npm run presentation
```
